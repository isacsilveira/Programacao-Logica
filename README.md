# Programação Lógica

Repositório contendo atividades, exercícios e trabalhos desenvolvidos durante a disciplina Programação Lógica da Universidade Federal de Alfenas (UNIFAL-MG).

## 📚 Sobre a disciplina

A disciplina de Programação Lógica tem como objetivo apresentar o paradigma da programação declarativa baseada em lógica, no qual os programas são construídos a partir de fatos, regras e relações, e a resolução dos problemas é feita por mecanismos de inferência e unificação.

Durante a disciplina foram desenvolvidas atividades práticas para aplicar os conceitos fundamentais da programação lógica na implementação de programas utilizando a linguagem **Prolog**.

## 📖 Conteúdo abordado

- Introdução à Programação Lógica
- Fatos, regras e consultas
- Unificação e resolução (backtracking)
- Estruturas de dados em Prolog (listas, átomos, termos)
- Recursão em Prolog
- Predicados e cláusulas
- Representação e manipulação de conhecimento
- Resolução de problemas com inferência lógica

## ▶️ Como executar

1. Instale um interpretador Prolog, como o [SWI-Prolog](https://www.swi-prolog.org/).
2. Clone o repositório:
   ```bash
   git clone https://github.com/isacsilveira/Programacao-Logica.git
   cd Programacao-Logica
   ```
3. Execute o arquivo no interpretador:
   ```bash
   swipl trab_Prolog.pl
   ```
